# Smart Parking System

A desktop-based parking reservation system developed using C#, Windows Forms, and SQLite.

---

## Overview

The Smart Parking System allows drivers to reserve parking slots while enabling administrators to manage parking operations through a dedicated dashboard.

---

## Features

- User Authentication
- Driver Dashboard
- Administrator Dashboard
- Parking Slot Reservation
- Automatic Reservation Expiry
- Automatic Slot Release
- SQLite Database Integration

---

## Technologies

- C#
- .NET Framework
- Windows Forms
- SQLite
